# Скрипт автоматического сбора данных (Парсер на Python)

Простой и эффективный парсер для сбора текстовых данных с веб-сайта и экспорта в структурированный Excel-отчет.

## Что делает скрипт:
- Обходит страницы сайта и извлекает целевые блоки данных.
- Связывает связанные сущности (Текст цитаты, Автор, Список тегов) внутри родительских контейнеров, исключая смещение строк.
- Форматирует списки тегов в аккуратную строку для удобного чтения в ячейке.
- Автоматически генерирует файл `quotes_portfolio.xlsx` без технических индексов.

## Что использовалось:
- **Python 3**
- **BeautifulSoup4 & LXML** (для быстрого разбора HTML-кода)
- **Requests** (для отправки сетевых запросов)
- **Pandas & Openpyxl** (для формирования и сохранения Excel-таблиц)
